# Netflix Recommendation Analytics System (SQL Project)

# Overview
Developed a SQL-based analytics system inspired by Netflix to analyze content trends, user preferences, and generate recommendations.

# Tools Used
- MySQL
- MySQL Workbench
- Power BI
- GitHub

# Features
- Designed relational database schema
- Imported real-world Netflix dataset from Kaggle
- Performed data analysis using SQL queries
- Used window functions for ranking and analytics
- Implemented stored procedures
- Optimized queries using indexes

# Insights
- Identified top content-producing countries
- Analyzed genre popularity
- Tracked content growth over years
- Built recommendation logic using SQL

## Project Highlights
- Advanced SQL (CTE, Window Functions, Subqueries)
- Real-world dataset usage
- Business-oriented insights
